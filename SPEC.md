# リフォーム費用計算【無料】キッチン・浴室・外壁の相場が分かる - 技術仕様書

## 概要

**サービス名**: Reform Cost Simulator
**バージョン**: 1.5.0
**更新日**: 2026-06-19
**URL**: https://appadaycreator.com/reform-cost-simulator/

リフォーム箇所・広さ・グレードを選択して費用相場を無料シミュレーション。補助金情報も掲載。
計算結果を自動保存・復元し、いつでも前回の結果を確認できます。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 主な機能

### 基本機能
- リフォーム箇所（キッチン、浴室、トイレ、床・壁、外壁塗装、屋根修繕、窓・断熱、バリアフリー化）の複数選択
- 建物タイプ（マンション / 一戸建て）、築年数、グレード（シンプル / スタンダード / こだわり）による費用計算
- 対象補助金・減税情報の自動表示

### 新機能（v1.2.0）
- **計算履歴の自動保存**: localStorage で最大 20 件の計算履歴を保存
- **履歴復元機能**: 「前回の計算結果を確認」から過去の計算をワンクリック復元
- **結果のコピー**: 計算結果をテキストでクリップボードにコピー
- **印刷機能**: 最適化された印刷プレビュー（不要な要素を非表示）
- **ダウンロード機能**: 計算結果をテキストファイルでダウンロード（ファイル名: reform-result-YYYYMMDD-HHMMSS.txt）

### 新機能（v1.3.0 - M4・M6実装）
- **M4【満足】グラフ・スコア・ゲージで結果可視化**: Chart.js で部位別費用を棒グラフで表示（低い場合・高い場合の比較）
- **M6【再訪×満足】目標設定・達成度トラッキング**: 目標予算の入力、達成率の計算・表示、プログレスバー・残額表示（localStorage で目標値を保持）

### 新機能（v1.4.0 - M8実装）
- **M8【再訪】ツールチップ・入力ガイダンス充実**: 
  - 各入力フィールドに info アイコン付きツールチップを追加
  - 建物の種類：マンション/戸建の工事費の違いを説明
  - 築年数：古い建物ほどリスク高いことを説明
  - 設備グレード：グレード別の費用差を説明
  - リフォーム箇所：複数選択の利点を説明
  - 目標予算：達成度の活用方法を説明
  - 各フィールド下部にヘルパーテキストを追加して使い方を案内

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 建物情報（タイプ、築年数、グレード）を選択する
3. リフォームしたい箇所を複数選択する
4. 「費用を計算する」ボタンをクリックして結果を表示
5. 結果画面で以下の操作が可能：
   - **コピー**: テキスト形式で結果をコピー
   - **印刷**: 最適化された形式で印刷
   - **DL**: テキストファイルをダウンロード
   - **LINEでシェア** / **Xでシェア**: SNS にシェア
   - **もう一度試す**: 入力フォームに戻る

## 履歴機能の活用

- 入力フォーム下部の「前回の計算結果を確認」ボタンをクリック
- 過去 20 件の計算結果が時系列で表示される
- 履歴アイテムをクリックで即座に復元（入力フォームと結果が同時に復元）
- 「履歴をすべてクリア」で全履歴削除可能

## よくある質問（FAQ）

**Q: リフォーム費用計算は無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 何回でも使えますか？**

はい、回数制限なく何度でもご利用いただけます。

**Q: 入力したデータはサーバーに送信されますか？**

いいえ。すべての処理はブラウザ内で完結し、入力内容はサーバーへ送信されません。

**Q: スマートフォンでも使えますか？**

はい、スマートフォン・タブレット・PCすべてに最適化されています。

**Q: 結果を保存・共有できますか？**

スクリーンショットでの保存またはSNSシェアボタンからご共有いただけます。
